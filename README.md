# Q_IntegrityTEST Streamlit App

Este repositorio contiene la aplicación Streamlit para Q_IntegrityTEST.

## Cómo ejecutar en Streamlit Cloud

1. Sube todos los archivos de la carpeta demoV3 a este repositorio.
2. Asegúrate de tener el archivo requirements.txt en la raíz.
3. El archivo principal debe ser app.py.

## Estructura
- app.py
- api_ia.py
- requirements.txt
- ...otros archivos

## Dependencias
Las dependencias están en requirements.txt.

## Notas
- Si usas archivos CSV, PDF, DOCX, etc., súbelos también.
- Para OCR, asegúrate de que pytesseract esté correctamente configurado en Streamlit Cloud.
